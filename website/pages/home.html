MPOOL is a Monacoin mining pool
  • Server: Japan
  • Pool fee is 0.1%
  • Payment method is prop
  • Payment is 100 confirm and minimum payment is 1 MONA
Getting Started
  • Download a miner. Best miner is here
  • ./ccminer -a -a lyra2v2 -o stratum+tcp://mona.mpool.jp:[PORT] -u [MONACOIN ADDRESS] -p x
  • PORT : 3333 : Hashrate ~1MH/s
    PORT : 4444 : Hashrate ~10MH/s
    PORT : 5555 : Hashrate ~100MH/s
    PORT : 6666 : Hashrate ~1GH/s
    PORT : 7777 : Hashrate ~10GH/s
    PORT : 8888 : Hashrate 10GH/s~
  • MONACOIN ADDRESS is your monacoin address
  • if workername add ./ccminer -a -a lyra2v2 -o stratum+tcp://mona.mpool.jp:[PORT] -u [MONACOIN ADDRESS].[WORKERNAME] -p x
